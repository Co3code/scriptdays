# JavaScript Mastery Challenge - 30 Days to Proficiency (Beginner)

## Overview

1. Read the lesson.
2. Complete `// TODO` exercises.
3. Run `node dayN.js` to test (uses self-checks).
4. Mark as done here when complete.
